How It Works

This demo shows how Arduino commands in text format (like <LED_ON>) are converted to efficient binary packets for transmission.

The binary protocol uses: START_BYTE (0xAA) + LENGTH + PAYLOAD (max 32 bytes) + CHECKSUM + END_BYTE (0xBB)
